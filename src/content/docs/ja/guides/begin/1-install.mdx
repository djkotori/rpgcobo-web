### <M code="f10d"/> RPG-Coboの動作環境
RPG-Coboをご使用の前に、お使いのコンピューターが動作環境を満たしているか確認してください。<br/>
**※通常は、Windows11が動作する全てのPCで問題なく使うことができます。**

|  |最低動作環境|推奨動作環境|
|--|-----------|-----------|
|OS|Microsoft® Windows10 (64bit)| |
|CPU|4コア以上のIntel又はAMD|Intel Core i5-8250U / AMD Ryzen3 7530|
|メモリ|8GB|16GB|
|GPU|OpenGL 4.2～|NVidia Geforce 1050 / AMD Radeon Vega 6|
|画面サイズ|1920x1080| |
|インターネット|必須||
