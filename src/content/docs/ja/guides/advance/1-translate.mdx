### <M code="f10d"/> ローカライズの重要性

世界的にゲームプレイヤー人口は増え続けており、ゲーム制作側でも多言語に対応したいという要求は高まってきています。    
しかし、ローカライズ作業はとても大変なものです。特に個人制作ではほとんど不可能と言っても良く、諦めてしまっている人も多いのではないでしょうか？

RPG-Coboを使えば、AI翻訳の力を借りてローカライズが驚くほど簡単に出来てしまいます！翻訳にかかる作業時間はものの数分にもならないでしょう。

もちろん、プロの翻訳家と同レベルの翻訳品質というわけにはいきませんが、皆さんの作ったゲームがとにかく翻訳されて
世界中のプレイヤーをターゲットにできるという事実は、何物にも代えがたい強みになるはずです。

### <M code="f10d"/> 対応しているAI翻訳
RPG-Coboは、ゲーム制作者の使い方に応じるために複数のAI翻訳方法に対応しています。

##### 1. Web翻訳を使ったAI翻訳（非商用のみ）
Google Web翻訳を使用した、お手軽で強力なAI翻訳です。  
まずはこちらを使ってAI翻訳の流れをつかみましょう。

##### 2. Google Translation API
Google Cloudの提供するAI翻訳サービスです。有料ですが商用利用も可能で翻訳品質も高いです。  
難点は、Google Cloudへのユーザー登録やAPIの有効化、APIキーの取得など面倒な手続きが多いことです。
料金については心配する必要はほぼ無いでしょう。

<Aside type="caution">
明確な規約が見当たらないのですが、Google Web翻訳は非営利のページのみ無料で利用できる模様です。[⇒こちらを参照](https://developers.google.com/search/blog/2020/05/google-translates-website-translator?hl=ja)  
フリーゲームを制作している方なら問題ないですが、販売を考えている方は Google Translation API を使えばライセンス上は安全です。  
**Google Translation API は登録手順は面倒ですが、無料枠でも十分な文字数の翻訳もできます。**
</Aside>


### <M code="f10d"/> ローカライズの流れ

<Steps>
1. アクティビティバーの「<M code="e8e2"/> 翻訳ツール」を開き、サイドメニュー左上の <M code="e5d2"/> ボタンから「<M code="e627"/> ソース翻訳辞書の作成」のメニューを選択します。
    <SS src='docs/translate-1.png' alt=''/>
2. データベースの各テキストが翻訳元の辞書に変換されます。「保存」ボタンでこの辞書を保存しましょう。
    <SS src='docs/translate-2.png' alt=''/>
3. 次に「ターゲット翻訳辞書」のリストから翻訳したい言語、ここでは英語で右クリックメニューの「<M code="e148"/> ターゲット翻訳辞書を作成」を選択します。
    <SS src='docs/translate-3.png' alt=''/>
4. ターゲット翻訳辞書を作成するダイアログが表示されます。**「自動翻訳の方法」は今回は「Web翻訳（非商用のみ）」を選択します。**
    <SS src='docs/translate-4.png' alt=''/>
5. Web自動翻訳の手順の通りにまずは「1.翻訳用のウェブページを開く」の「開く」ボタンを押してください。ブラウザが立ち上がって翻訳用のページが開きます。

6. ブラウザで「言語の選択▼」メニューから「英語」を選択します。すると自動翻訳が走り、表示されている辞書が英語に翻訳されます。
    <SS src='docs/translate-5.png' alt=''/>

7. 「下にスクロール」のボタンを押すと、ページが下までスクロールします。そして、そこにある「翻訳結果をクリップボードにコピー」のボタンを押します。  
    <SS src='docs/translate-6.png' alt=''/>
    <SS src='docs/translate-7.png' alt=''/>
   これでブラウザ側の操作は終了です。ツールに戻りましょう。

8. 「2.翻訳結果をクリップボードからインポート」の「インポート」ボタンを押してください。確認のダイアログが出るので、「保存」ボタンで決定します。

9. これで完成です！自動翻訳された辞書に対してさらに手動で編集することもできます。次の説明に移りましょう。
</Steps>

### <M code="f10d"/> 翻訳辞書を手動で編集する

「ターゲット翻訳辞書」を生成した言語は、ダブルクリックで編集することができます。

<div class="sspart">
<SS src='docs/translate-8.png' alt=''/>

##### <M part="A"/> 表示フィルター
表示するワードリストを絞り込むための条件を指定します。

##### <M part="B"/> メニューボタン
このメニューから、TSVのコピー＆ペーストができます。使い慣れたExcelやスプレッドシート等のツールで翻訳作業をしたり、
外部の翻訳家にデータを受け渡しするときに使用します。

##### <M part="C"/> ワードリスト
- ワードキー: データベースのどの項目のワードかを識別するIDです。
- ソースワード: 翻訳前の文
- ターゲットワード: 翻訳後の文（これを編集）
- 完了チェック: 翻訳が完了したワードはここにチェックを入れましょう。サイドメニューに表示される進捗率が上がっていきます。
</div>


### <M code="f10d"/> ローカライズのデバッグ

環境設定の「デバッグ」タブ内の「デバッグする言語」を指定すると、言語を切り替えてゲームをデバッグすることができます。
