## Math（数学関数）

数学関数は`Math`テーブルに格納されています。

### 定数

```javascript
Math.PI      // 円周率（約3.14159）
Math.DTOR    // 度→ラジアン変換係数（PI / 180）
Math.RTOD    // ラジアン→度変換係数（180 / PI）
Math.PI2     // 円周率×2
Math.HPI     // 円周率÷2
```

### 基本数学関数

#### `Math.abs(value)`
絶対値を返します。

**引数**:
- `value` (int/float): 入力値

**戻り値**: 絶対値

#### `Math.sqrt(value)`
平方根を返します。

**引数**:
- `value` (float): 入力値

**戻り値**: 平方根

#### `Math.saturate(value)`
値を0～1の範囲にクランプします。

**引数**:
- `value` (float): 入力値

**戻り値**: 0～1の範囲にクランプされた値

#### `Math.sign(value)`
符号を返します（-1, 0, 1）。

**引数**:
- `value` (int/float): 入力値

**戻り値**: -1（負）、0（ゼロ）、1（正）

### 三角関数

#### `Math.sin(rad)`, `Math.cos(rad)`, `Math.tan(rad)`
三角関数。

**引数**:
- `rad` (float): ラジアン

**戻り値**: 三角関数の値

#### `Math.asin(value)`, `Math.acos(value)`, `Math.atan(value)`
逆三角関数。

**引数**:
- `value` (float): 入力値

**戻り値**: ラジアン

**説明**: `asin`と`acos`は範囲外の値（-1～1）を自動的にクランプします。

#### `Math.atan2(y, x)`
2引数版のアークタンジェント。

**引数**:
- `y` (float): Y成分
- `x` (float): X成分

**戻り値**: ラジアン（-π～π）

### 対数・指数関数

#### `Math.log(value)`, `Math.log2(value)`, `Math.log10(value)`
対数関数（自然対数、底2、底10）。

**引数**:
- `value` (float): 入力値

**戻り値**: 対数値

#### `Math.logn(value, base)`
任意の底の対数。

**引数**:
- `value` (float): 入力値
- `base` (float): 底

**戻り値**: 対数値

#### `Math.exp(value)`, `Math.exp2(value)`
指数関数（e^x、2^x）。

**引数**:
- `value` (float): 指数

**戻り値**: 指数値

#### `Math.pow(base, exp)`
累乗計算。

**引数**:
- `base` (float): 底
- `exp` (float): 指数

**戻り値**: base^exp

### 丸め関数

#### `Math.floor(value)`, `Math.ceil(value)`, `Math.round(value)`
丸め関数（切り捨て、切り上げ、四捨五入）。

**引数**:
- `value` (float): 入力値

**戻り値**: 丸められたfloat値

#### `Math.ifloor(value)`, `Math.iceil(value)`, `Math.iround(value)`
整数として丸めます。

**引数**:
- `value` (float): 入力値

**戻り値**: 丸められたint値

#### `Math.floor2(value)`
2の累乗で切り捨て。

**引数**:
- `value` (float): 入力値

**戻り値**: 2の累乗で切り捨てられた値

### 最大・最小・範囲

#### `Math.max(value1, value2, ...)`
最大値を返します（可変長引数）。

**引数**:
- `value1, value2, ...` (int/float): 比較する値

**戻り値**: 最大値

#### `Math.min(value1, value2, ...)`
最小値を返します（可変長引数）。

**引数**:
- `value1, value2, ...` (int/float): 比較する値

**戻り値**: 最小値

#### `Math.clamp(value, min, max)`
値を指定範囲にクランプします。

**引数**:
- `value` (int/float): 入力値
- `min` (int/float): 最小値
- `max` (int/float): 最大値

**戻り値**: クランプされた値

### 補間

#### `Math.lerp(a, b, t)`
線形補間。

**引数**:
- `a` (float): 開始値
- `b` (float): 終了値
- `t` (float): 補間係数（0～1）

**戻り値**: 補間された値（a + (b-a)*t）

#### `Math.lerpcolor(color1, color2, t)`
色の線形補間。

**引数**:
- `color1` (int): 開始色（RGBA形式）
- `color2` (int): 終了色（RGBA形式）
- `t` (float): 補間係数（0～1）

**戻り値**: 補間された色（int）

### ビット演算

#### `Math.shift(value, shift)`
ビットシフト。

**引数**:
- `value` (int): 値
- `shift` (int): シフト量（正：左シフト、負：右シフト）

**戻り値**: シフトされた値

#### `Math.bitcount(value)`
セットされているビット数を返します。

**引数**:
- `value` (int): 値

**戻り値**: 1になっているビットの数

#### `Math.pot(value)`
次の2の累乗を返します。

**引数**:
- `value` (int): 入力値

**戻り値**: value以上の最小の2の累乗

#### `Math.msb(value)`
最上位ビットの位置を返します。

**引数**:
- `value` (int): 値

**戻り値**: 最上位ビットのインデックス

#### `Math.lsb(value)`
最下位ビットの位置を返します。

**引数**:
- `value` (int): 値

**戻り値**: 最下位ビットのインデックス

### その他

#### `Math.mod2(value, min, max)`
範囲内にループする剰余。

**引数**:
- `value` (float): 値
- `min` (float): 最小値
- `max` (float): 最大値

**戻り値**: min～maxの範囲にループされた値

#### `Math.digit(value, precision)`
指定精度に丸めます。

**引数**:
- `value` (float): 値
- `precision` (int): 精度

**戻り値**: 丸められた値

#### `Math.vtohp(x, y, z)` / `Math.vtohp(vector)` / `Math.vtohp(array)`
ベクトルから方位角・仰角を計算します。

**引数**:
- `x, y, z` (float): ベクトル成分
- または `vector` (Vector3D): ベクトル
- または `array` (array): `[x, y, z]`の配列

**戻り値**: `[heading, pitch]`のタプル（ラジアン）

#### `Math.length(...values)`
複数の値から長さを計算します。

**引数**:
- `...values` (float): 各成分の値

**戻り値**: `sqrt(sum(values))`

## クラス

### Random

疑似乱数生成器（Xorshift）。

#### コンストラクタ

```javascript
Random([seed[, warmup]])
```

**引数**:
- `seed` (int, オプション): シード値（省略時は現在時刻）
- `warmup` (int, オプション): ウォームアップ回数

#### フィールド

- `x`, `y`, `z`, `w` (int): 内部状態（読み取り専用）

#### メソッド

##### `setSeed(seed[, warmup])`
シード値を設定します。

**引数**:
- `seed` (int): シード値
- `warmup` (int, オプション): ウォームアップ回数

##### `next([bits])`
ランダムビットを生成します。

**引数**:
- `bits` (int, オプション): ビット数（デフォルト: 24）

**戻り値**: ランダムな整数

##### `nexti([min,] max)`
整数範囲のランダム値を生成します。

**引数**:
- `min` (int, オプション): 最小値（省略時: 0）
- `max` (int): 最大値（**含まない**）

**戻り値**: min以上max未満のランダムな整数

##### `nextf([min,] max)`
浮動小数点範囲のランダム値を生成します。

**引数**:
- `min` (float, オプション): 最小値（省略時: 0.0）
- `max` (float, オプション): 最大値（省略時: 1.0）

**戻り値**: min以上max未満のランダムなfloat

##### `crypt(buffer[, length])`
バッファを乱数でXOR暗号化します。

**引数**:
- `buffer` (ByteBuffer): 暗号化するバッファ
- `length` (int, オプション): 暗号化する長さ

##### `choose(value1, value2, ...)`
引数からランダムに1つ選びます。

**引数**:
- `value1, value2, ...`: 選択肢

**戻り値**: ランダムに選ばれた値

##### `choosea(array)`
配列の要素からランダムに1つ選びます。

**引数**:
- `array` (array): 配列

**戻り値**: ランダムに選ばれた要素

#### 呼び出し演算子

```javascript
rnd(max)              // 0～max
rnd(min, max)         // min～max
```

**説明**: `nexti`または`nextf`を呼び出します（引数の型に応じて自動判別）。

### Vector3D

3次元ベクトルクラス。

#### コンストラクタ

```javascript
Vector3D()                          // (0, 0, 0)
Vector3D(value)                     // (value, value, value)
Vector3D(x, y)                      // (x, y, 0)
Vector3D(x, y, z)                   // (x, y, z)
Vector3D(vector)                    // コピー
Vector3D([x, y, z])                 // 配列から
```

#### フィールド

- `x`, `y`, `z` (float): 各成分
- `[0]`, `[1]`, `[2]` (float): インデックスアクセス（エイリアス）

#### メソッド

##### 設定・取得

##### `set(x, y, z)` / `set(value)` / `set(vector)` / `set([x,y,z])`
値を設定します。

**戻り値**: 自身

##### `iszero()`
ゼロベクトルかどうか判定します。

**戻り値**: すべての成分が0の場合`true`

##### `equals(other)`
別のベクトルと等しいか判定します。

**引数**:
- `other` (Vector3D): 比較対象

**戻り値**: 等しい場合`true`

##### 算術演算

##### `add(x, y, z)` / `add(value)` / `add(vector)`
加算します。

**戻り値**: 自身

##### `sub(x, y, z)` / `sub(value)` / `sub(vector)`
減算します。

**戻り値**: 自身

##### `mul(x, y, z)` / `mul(value)` / `mul(vector)`
乗算します。

**戻り値**: 自身

##### `div(x, y, z)` / `div(value)` / `div(vector)`
除算します。

**戻り値**: 自身

##### `scale(x, y, z)` / `scale(value)` / `scale(vector)`
スケール（乗算のエイリアス）。

**戻り値**: 自身

##### ベクトル演算

##### `length()`
ベクトルの長さを返します。

**戻り値**: `sqrt(x*x + y*y + z*z)`

##### `lengthSq()`
ベクトルの長さの2乗を返します。

**戻り値**: `x*x + y*y + z*z`

##### `normalize()`
正規化します（長さを1にする）。

**戻り値**: 自身

##### `dot(other)`
内積を計算します。

**引数**:
- `other` (Vector3D): 別のベクトル

**戻り値**: 内積値

##### `cross(other[, result])`
外積を計算します。

**引数**:
- `other` (Vector3D): 別のベクトル
- `result` (Vector3D, オプション): 結果の格納先

**戻り値**: 外積ベクトル（新規または`result`）

##### `hypot(other)`
2つのベクトル間の距離を返します。

**引数**:
- `other` (Vector3D): 別のベクトル

**戻り値**: 距離

##### `hypotSq(other)`
2つのベクトル間の距離の2乗を返します。

**引数**:
- `other` (Vector3D): 別のベクトル

**戻り値**: 距離の2乗

##### 変換

##### `transform(matrix)`
行列で変換します（平行移動を含む）。

**引数**:
- `matrix` (Matrix): 変換行列

**戻り値**: 自身

##### `transform0(matrix)`
行列で変換します（平行移動を含まない）。

**引数**:
- `matrix` (Matrix): 変換行列

**戻り値**: 自身

##### `transformw(matrix)`
行列で変換し、wコンポーネントを返します。

**引数**:
- `matrix` (Matrix): 変換行列

**戻り値**: w成分（float）

##### `rotate(quaternion)` / `rotate(h, p, b)` / `rotate(vector)`
回転を適用します。

**引数**:
- `quaternion` (Quaternion): 回転クォータニオン
- または `h, p, b` (float): heading, pitch, bank（ラジアン）
- または `vector` (Vector3D): 回転ベクトル

**戻り値**: 自身

##### その他

##### `saturate()`
各成分を0～1にクランプします。

**戻り値**: 自身

##### `floor()`, `ceil()`, `round()`
各成分を丸めます。

**戻り値**: 自身

##### `lerp(other, t[, result])`
線形補間します。

**引数**:
- `other` (Vector3D): 終了ベクトル
- `t` (float): 補間係数（0～1）
- `result` (Vector3D, オプション): 結果の格納先

**戻り値**: 補間されたベクトル

##### `min(other)`, `max(other)`
各成分の最小・最大をとります。

**引数**:
- `other` (Vector3D): 別のベクトル

**戻り値**: 自身

##### `hasNaN()`
NaNを含むかチェックします。

**戻り値**: NaNを含む場合`true`

##### `nanCheck()`
NaN/Inf情報をビットフィールドで返します。

**戻り値**: ビットフィールド（int）

#### 静的メソッド

##### `Vector3D.fromRandom(min, max)`
ランダムなベクトルを生成します。

**引数**:
- `min` (float): 各成分の最小値
- `max` (float): 各成分の最大値

**戻り値**: ランダムなベクトル

#### 演算子

```javascript
v1 + v2          // 加算（新規ベクトル）
v1 - v2          // 減算（新規ベクトル）
v1 * v2          // 乗算（新規ベクトル）
v1 / v2          // 除算（新規ベクトル）
v1 % v2          // 剰余（新規ベクトル）
v1 << v2         // 代入（v1 = v2）
-v               // 反転（新規ベクトル）
v1 | v2          // 内積（float）
v1 ^ v2          // 外積（新規ベクトル）
v(x, y, z)       // 呼び出し演算子で設定
```

### Quaternion

四元数（回転）クラス。

#### コンストラクタ

```javascript
Quaternion()                              // 単位クォータニオン
Quaternion(x, y, z, w)                    // 直接指定
Quaternion(angle, axis)                   // 軸まわりの回転
Quaternion(quaternion)                    // コピー
Quaternion(matrix)                        // 行列から
Quaternion([x, y, z, w])                  // 配列から
```

#### フィールド

- `x`, `y`, `z`, `w` (float): 各成分
- `[0]`, `[1]`, `[2]`, `[3]` (float): インデックスアクセス

#### メソッド

##### `set(x, y, z, w)` / `set(angle, axis)` / `set(other)`
値を設定します。

**戻り値**: 自身

##### `iszero()`
単位クォータニオンかどうか判定します。

**戻り値**: 単位クォータニオンの場合`true`（w=1, x=y=z=0）

##### `equals(other)`
別のクォータニオンと等しいか判定します。

**引数**:
- `other` (Quaternion): 比較対象

**戻り値**: 等しい場合`true`

##### `norm()`
ノルム（長さ）を返します。

**戻り値**: ノルム

##### `normalize()`
正規化します。

**戻り値**: 自身

##### `toMatrix()`
3x3行列に変換します。

**戻り値**: `Matrix`オブジェクト

##### `lerp(other, t[, result])`, `slerp(other, t[, result])`
線形補間・球面線形補間します。

**引数**:
- `other` (Quaternion): 終了クォータニオン
- `t` (float): 補間係数（0～1）
- `result` (Quaternion, オプション): 結果の格納先

**戻り値**: 補間されたクォータニオン

##### `synthesize(other)`
クォータニオンを合成します（乗算）。

**引数**:
- `other` (Quaternion): 合成するクォータニオン

**戻り値**: 自身

##### `getHPB([result])`
Heading-Pitch-Bank角を取得します。

**引数**:
- `result` (Vector3D, オプション): 結果の格納先

**戻り値**: `Vector3D(h, p, b)`

##### `setHPB(h, p, b)` / `setHPB(vector)`
Heading-Pitch-Bank角から設定します。

**引数**:
- `h, p, b` (float): Heading, Pitch, Bank（ラジアン）
- または `vector` (Vector3D): HPBベクトル

**戻り値**: 自身

##### `setAxis(angle, x, y, z)`
軸まわりの回転を設定します。

**引数**:
- `angle` (float): 回転角（ラジアン）
- `x, y, z` (float): 回転軸

#### 静的メソッド

##### `Quaternion.fromAxis(angle, axis)`
軸まわりの回転クォータニオンを作成します。

**引数**:
- `angle` (float): 回転角（ラジアン）
- `axis` (Vector3D): 回転軸

**戻り値**: `Quaternion`オブジェクト

##### `Quaternion.from2Axis(from, to)`
2つのベクトル間の回転を作成します。

**引数**:
- `from` (Vector3D): 開始ベクトル
- `to` (Vector3D): 終了ベクトル

**戻り値**: `Quaternion`オブジェクト

##### `Quaternion.fromHPB(h, p, b)` / `Quaternion.fromHPB(vector)` / `Quaternion.fromHPB([h,p,b])`
HPB角からクォータニオンを作成します。

**引数**:
- `h, p, b` (float): Heading, Pitch, Bank
- または `vector` (Vector3D)
- または `[h, p, b]` (array)

**戻り値**: `Quaternion`オブジェクト

#### 演算子

```javascript
q1 * q2          // 合成（新規クォータニオン）
q * v            // ベクトル回転（新規ベクトル）
q1 << q2         // 代入（q1 = q2）
-q               // 共役（新規クォータニオン）
q(...)           // 呼び出し演算子で設定
```

### Plane

平面クラス（ax + by + cz + d = 0）。

#### コンストラクタ

```javascript
Plane()                        // (0, 0, 0, 1)
Plane(a, b, c, d)              // 直接指定
Plane(plane)                   // コピー
Plane([a, b, c, d])            // 配列から
```

#### フィールド

- `a`, `b`, `c`, `d` (float): 平面の係数
- `[0]`, `[1]`, `[2]`, `[3]` (float): インデックスアクセス

#### メソッド

##### `set(a, b, c, d)` / `set(plane)`
値を設定します。

**戻り値**: 自身

##### `equals(other)`
別の平面と等しいか判定します。

**引数**:
- `other` (Plane): 比較対象

**戻り値**: 等しい場合`true`

##### `normalize()`
正規化します（法線ベクトルを長さ1に）。

**戻り値**: 自身

##### `lerp(other, t)`
線形補間します。

**引数**:
- `other` (Plane): 終了平面
- `t` (float): 補間係数

**戻り値**: 新しい`Plane`オブジェクト

##### `setParam(nx, ny, nz, px, py, pz)` / `setParam(normal, position)`
法線と位置から設定します。

**引数**:
- `nx, ny, nz` (float): 法線ベクトル
- `px, py, pz` (float): 平面上の位置
- または `normal`, `position` (Vector3D)

**戻り値**: 自身

##### `getHeight(x, y, z)` / `getHeight(position)`
指定点の平面からの高さを返します。

**引数**:
- `x, y, z` (float): 位置
- または `position` (Vector3D)

**戻り値**: 高さ（float、符号付き距離）

##### `inside(x, y, z)` / `inside(vector)` / `inside(rect3d)`
指定オブジェクトが平面の内側かどうか判定します。

### bigint

多倍長整数クラス。

#### コンストラクタ

```javascript
bigint()                                 // 0
bigint(value)                            // 整数から
bigint(string[, radix])                  // 文字列から（10進または16進）
bigint(buffer[, offset, length])         // バイトバッファから
```

**引数**:
- `value` (int/long): 初期値
- `string` (string): 10進または16進文字列
- `radix` (int, オプション): 基数（16で16進）
- `buffer` (ByteBuffer): バイトバッファ
- `offset`, `length` (int, オプション): 読み取り範囲

#### フィールド

- `sign` (int): 符号（読み取り専用）
- `size` (int): サイズ（内部配列の要素数、読み取り専用）
- `[index]` (int): 内部配列の要素（読み取り専用）

#### メソッド

##### `add(other)`, `sub(other)`, `mul(other)`, `div(other)`, `mod(other)`
算術演算を行います。

**引数**:
- `other` (bigint): 演算対象

**戻り値**: 自身

**説明**: 自身を変更します（破壊的操作）。

##### `powmod(p, n)`
べき乗剰余を計算します（this^p mod n）。

**引数**:
- `p` (bigint): 指数
- `n` (bigint): 法

**戻り値**: 自身

##### `tostring([radix])`
文字列に変換します。

**引数**:
- `radix` (int, オプション): 16で16進（デフォルトは10進）

**戻り値**: 文字列

**引数**:
- `buffer` (ByteBuffer, オプション): 書き込み先バッファ

## 注意事項

1. **座標系**: SakanaGLは右手座標系を使用しています。

2. **角度の単位**: すべての角度はラジアンで指定します。度からラジアンへの変換には`Math.DTOR`を使用してください。

3. **クォータニオン**: 回転の合成は右から左へ（`q1 * q2`は`q2`の後に`q1`を適用）。

4. **行列**: 列優先で格納されています。OpenGLと互換性があります。

5. **Transform**: `Transform + Transform`は階層的な変換の合成を行います。

6. **バウンディングボックス**: `Rect3D`は中心座標と半径で表現されます（最小・最大座標ではない）。

7. **カメラ**: `updatePlanes()`を呼び出さないと`isInside()`は正しく動作しません。

8. **演算子オーバーロード**: `+`, `-`, `*`, `/`, `|`, `^`, `<<`などの演算子が多くのクラスでオーバーロードされています。

9. **破壊的操作**: `add()`, `sub()`, `mul()`などのメソッドは自身を変更します。新しいオブジェクトを作成するには演算子（`+`, `-`, `*`）を使用してください。
