## <M code="f083"/> ご注意
お問い合わせの方は以下の注意点をお読みになり、ご了承のうえお問い合わせください。

- このページはRPG-Coboの製品に対してサポートを行うための窓口ではありません。
  ツールの操作方法、動作不具合等のご質問には対応できません。
- ご意見、ご要望につきましては、個別には返信いたしません。
- お問い合わせに対してはメールで返信いたします。メールが受信できるよう予めご確認ください。

## <M code="e160"/> 問題が起きたときは

- よくある質問 : まずは、[よくある質問](/faq)の「問題が起きたときは」のタブを確認してみてください。
- ドキュメント : ドキュメントの「困ったときには」の章で解決方法があるか調べてみてください。
- Discord : Discordの#help-ja(質問)又は#bug-report-ja(バグ報告)チャンネルで報告してみてください。
  ※投稿の前に、他の投稿ですでに解決策があるかを予め確認してください。
- Github : 問題の改善に協力いただけるなら、GithubのIssueに報告をお願いします。
